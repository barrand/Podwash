# Slice 07 analysis-pipeline fixtures — provenance

> Named `analysis-provenance.md` (not `README.md` or `PROVENANCE.md`) on purpose: the Xcode
> `PBXFileSystemSynchronizedRootGroup` flattens bundled resources into the test
> bundle root, so duplicate basenames collide at build time (see `asr-README.md`,
> `transcripts/PROVENANCE.md`).

**Hard QA rule:** these goldens have provenance **independent of the code under
test**. They were hand-computed from `docs/specs/matching-spec.md` §3–§6 on
2026-07-09. They were **NOT** generated by `AnalysisPipeline` (the code they
verify). Regenerating a golden from the implementation it tests would make the
assertion circular and is forbidden.

Comparison tolerance: **±0.0005 s** for fast tests; **±200 ms** for slow tests
(live ASR drift budget per ADR-003).
